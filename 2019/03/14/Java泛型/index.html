Java泛型

发表于 2019-03-14 | 更新于 2019-09-14

泛型是从 Java 5 开始增加的对于数据类型的规范和检测机制，使用泛型有如下两个优点

  1. 可以有效地避免 ClassCastException ，在编译期就可以使用泛型发现类型转换错误，或者类型不匹配
  2. 避免强类型转换
